# App Updater System (Система управления обновлениями)

Кроссплатформенная REST API система для автоматизированного управления версиями и обновлениями клиентских приложений (Android, iOS, Windows, macOS, Linux).

* Язык: Java 17
* Фреймворк: Spring Boot 3
* База данных: PostgreSQL
* Безопасность: Spring Security + Stateless JWT (Cookies)
* Миграции/ORM: Hibernate / Spring Data JPA
* Документация: Swagger / OpenAPI 3.0
* Сборка: Maven

## Ключевые возможности
* Полный CRUD для версий приложений с учетом платформ.
* Определение типа обновления (`MANDATORY`, `OPTIONAL`, `DEPRECATED`).
* Отслеживание актуальных версий на устройствах пользователей (`lastSeen`).
* Тепловая карта распространения версий и статистика успешных обновлений.
* Автоматические уведомления в чат о выходе критических (MANDATORY) обновлений в Telegram.
* Кастомная система авторизации (Role-Based Access Control) с передачей JWT-токенов через защищенные `httpOnly` Cookies.
* Экспорт полного списка версий в формате CSV.
* Загрузка списка заблокированных (устаревших) версий через конфигурацию `deprecated.yml`.
